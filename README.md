# group_intro

## 포함 내용
1. 팀/팀원 소개 및 MBTI (구현)
2. 각자의 장점 (TMI에 작성으로 구현)
3. 자신의 협업 스타일 (TMI에 작성으로 구현)
4. 우리 팀만의 특징과 추구하는 궁극적 목표

6. 팀원 주소 (구현)
